# 🩺 WellCheck - Medical Advice System

WellCheck is a Java Swing-based desktop application that allows users to enter their symptoms and receive basic health recommendations along with educational information about common medical conditions.

⚠️ **Disclaimer:** This application is for educational purposes only and does not replace professional medical advice, diagnosis, or treatment.

---

## ✨ Features

- Modern and responsive Java Swing GUI
- Symptom-based condition detection using keyword matching
- Provides general medical recommendations
- Interactive learning cards for detected conditions
- Detailed information pop-ups for each condition
- Clean and user-friendly interface with custom styling
- Scrollable and responsive analysis section

---

## 🩺 Supported Conditions

Currently, WellCheck can analyze symptoms related to:

- Fever
- Cough
- Headache
- Common Cold
- Stomachache
- Sore Throat

---

## 🚀 How It Works

1. Enter your symptoms in the text area.
2. Click **Analyze Symptoms**.
3. The system:
   - Detects possible conditions using keyword matching.
   - Generates general health recommendations.
   - Displays educational information about the detected conditions.
4. Click **Learn More** to view detailed information about any condition.

## ▶️ Getting Started

### Prerequisites
- Java JDK 8 or above
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code)

### Installation

```bash
git clone https://github.com/MokshFF/WellCheck-Health-Assistant.git
cd WellCheck-Health-Assistant
javac WellCheck.java
java WellCheck
```
